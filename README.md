# at-translate

Automated translate the game [Andor's Trails](https://andorstrail.com/) into other languages.
Powered by OpenAI's GPT API and Weblate's format.

# How-To

## Environment

```python
pip install -r requirements.txt
# or
rye install
```

## Run Scripts

```python
python src/main.py
# or 
make # CMake is required
```
